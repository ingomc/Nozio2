# Nozio v0.6.0

## Highlights
- Einstellungen zeigen jetzt App-Infos inklusive Version und Buildnummer.
- Neues In-App-Update ueber GitHub Releases: Update pruefen, APK herunterladen und Installation starten.

## Improvements
- Automatische Update-Pruefung beim Oeffnen der Einstellungen.
- Manueller `Jetzt pruefen`-Button fuer Update-Checks.
- Update-Dialog mit Release-Titel und gekuerzten Release Notes.
- APK-Asset-Auswahl verbessert: `universal` wird bevorzugt, sonst die erste `.apk`.
- Fallback auf die GitHub-Release-Seite, falls kein direkter APK-Download moeglich ist.
- Flow fuer `Unbekannte Quellen` integriert, wenn fuer die APK-Installation erforderlich.

## Notes
- Android-Version auf `0.6.0` angehoben.
