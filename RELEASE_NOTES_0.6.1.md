# Nozio v0.6.1

## Highlights
- Einstellungen zeigen jetzt App-Infos inkl. Version und Buildnummer.
- Neues In-App-Update ueber GitHub Releases: Update pruefen, APK laden und Installation starten.

## Improvements
- Automatische Update-Pruefung beim Oeffnen der Einstellungen.
- Manueller `Jetzt pruefen`-Button fuer Update-Checks.
- Update-Dialog mit Release-Titel und gekuerzten Release-Notizen.
- APK-Asset-Auswahl verbessert: `universal` wird bevorzugt, sonst erste `.apk`.
- Fallback auf die GitHub-Release-Seite, falls kein direkter APK-Download moeglich ist.
- Unbekannte-Quellen-Flow integriert (`Unknown App Sources`), wenn fuer APK-Installation erforderlich.

## Notes
- Android-Version auf `0.6.1` angehoben.
