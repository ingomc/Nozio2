# Nozio v0.1.0

## Highlights
- Dashboard improved with smoother Material-style tab transitions.
- Search UI refinements:
  - pill-shaped search field
  - circular QR scanner action
  - improved color consistency
- Add food flow styling consistency updates.
- Profile improvements:
  - macro goals compact 3-column layout
  - save button enabled only when values changed
  - live calorie estimate from macro targets
- Daily tracking improvements:
  - steps entry moved to fast bottom sheet workflow
  - weight entry moved to fast bottom sheet workflow
  - daily weight history now tracked and visualized in profile chart
  - selectable weight chart ranges and horizontal scrolling for dense data
