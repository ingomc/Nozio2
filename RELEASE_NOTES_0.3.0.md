# Nozio v0.3.0

## Highlights
- Quick Add fuer freie Kalorien-Eintraege direkt aus der Suche.
- Eigenes Produkt anlegen inklusive Makros, Portion und Packung.
- Eigene Produkte werden nach dem Speichern sofort wieder ueber die App-Suche auffindbar.
- Search-Flow erweitert um schnelle Aktionen fuer manuelle Eintraege.

## Improvements
- Neue API-Route fuer benutzerdefinierte Produkte.
- Seed-Handling im Food-API verbessert, damit Basisdaten und benutzerdefinierte Produkte gemeinsam erhalten bleiben.
- Such- und Repository-Modelle unterstuetzen jetzt unterschiedliche Food-Quellen sauber, inklusive `CUSTOM`.
- Bottom-Sheet-Hoehen fuer grosse Eingabeformulare begrenzt, damit der Drag-Handle nicht in die Notch laeuft.

## Notes
- Android-Version auf `0.3.0` angehoben.
- Release Notes werden ab jetzt pro Release als eigene Datei gepflegt.
